# 📚 StudySphere

StudySphere is a collaborative study platform that helps students stay consistent, build study habits, and learn together.

## Features

- User Authentication (Register/Login)
- Create and Join Study Rooms
- Topic-Based Study Discussions
- Real-Time Room Conversations
- User Profiles
- Study Session Tracking
- Daily Study Streaks
- Longest Streak Tracking
- Study Streak Leaderboard

## How It Works

1. Users create or join study rooms.
2. Start a study session.
3. Complete a session after studying.
4. Build and maintain daily streaks.
5. Compete with other learners on the leaderboard.
